# HTML (HyperText Markup Language)

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages.

HTML defines the structure of a webpage using different elements called **tags**.

It is **not a programming language**. It is a **markup language**.

---

# What is HTML?

HTML stands for **HyperText Markup Language**.

It is used to:

- Create Web Pages
- Structure Content
- Display Text, Images, Videos and Links
- Build the foundation of every website

HTML works together with:

- CSS → Styling
- JavaScript → Functionality

---

# Why HTML?

Without HTML, a browser does not know how to display content.

HTML provides the basic structure of every webpage.

Example:

```
Website

↓

HTML
(Structure)

↓

CSS
(Design)

↓

JavaScript
(Functionality)
```

# Structure of an HTML Document

```html
<!DOCTYPE html>
<html>

<head>
    <title>My First Page</title>
</head>

<body>

    <h1>Hello World</h1>

</body>

</html>
```

# Basic HTML Tags

| Tag | Purpose |
|------|---------|
| `<html>` | Root Element |
| `<head>` | Contains metadata |
| `<title>` | Page Title |
| `<body>` | Visible Content |
| `<h1>` to `<h6>` | Headings |
| `<p>` | Paragraph |
| `<br>` | Line Break |
| `<hr>` | Horizontal Line |
| `<img>` | Image |
| `<a>` | Hyperlink |
| `<div>` | Block Container |
| `<span>` | Inline Container |

# Important Notes

- HTML is a Markup Language.
- HTML is not case-sensitive.
- Most tags have opening and closing tags.
- HTML provides only structure.
- CSS provides design.
- JavaScript provides functionality.
